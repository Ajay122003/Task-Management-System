#  Task Management System

##  Overview
This is a **Task Management Web Application** built using **Django**.  
It helps users to efficiently manage their daily tasks — allowing them to **create, update, delete, and mark tasks as completed or pending**.  
Each user has their **own personal dashboard**, ensuring privacy and smooth task handling.

---

##  Features
-  **Admin ** – Register, Login, Logout, visit tasks and users
-  **User Authentication** – Register, Login, Logout  
-  **Task CRUD Operations** – Add, Edit, Delete tasks  
-  **Mark as Completed / Pending**  
-  **Filter tasks by status** (All / Pending / Completed)  
-  **User-specific Data** – Each user sees only their tasks  
-  **Success/Error Messages** for all actions  
-  **Responsive UI** using Bootstrap  

---

| Component | Technology |
